# Проект: Интерактивный дашборд данных Gaia DR3
**Автор:** Парвиз (Parvizi Orzukul)
**Цель:** Визуализация астрофизических параметров звезд на основе данных космического телескопа Gaia.

## Описание данных
В проекте используются реальные данные миссии Gaia (Data Release 3). 
Датасет состоит из 3 связанных таблиц (связь по ключу `source_id`):
1. `gaia_main`: Координаты и яркость.
2. `gaia_params`: Температура и физические параметры.
3. `gaia_variability`: Скорости звезд.

Источник данных: [Gaia Archive](https://gea.esac.esa.int/archive/)

## Инструменты
* **DuckDB**: Локальная база данных для хранения и SQL-запросов.
* **Pandas**: Обработка данных.
* **Plotly**: Интерактивные графики.
* **Streamlit**: Веб-интерфейс дашборда.

## Структура репозитория
* `source/`: Исходные CSV файлы.
* `queries/`: SQL-скрипты для обработки данных.
* `ddl.py`: Скрипт инициализации БД.
* `db.py`: Модуль доступа к данным.
* `main.py`: Код дашборда.
* `my.db`: Файл базы данных.

## Запуск проекта
1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте БД: `python3 ddl.py`
3. Запустите дашборд: `streamlit run main.py`
